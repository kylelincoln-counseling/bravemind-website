Meta Quest Link PC System Requirements

To use Meta Quest Link or Air Link to connect your Meta Quest headset to a Windows PC, ensure your computer meets the following specifications.

Minimum PC Specifications

  • Operating System: Windows 10 or Windows 11
  • Processor: Intel Core i5-4590 / AMD Ryzen 5 1500X or greater
  • Memory: 8 GB RAM or more
  • Graphics Card: NVIDIA GeForce GTX 970 / AMD Radeon R9 290 or greater
  • USB Port: 1x USB 3.0 port

Recommended PC Specifications

  • Processor: Intel Core i7 / AMD Ryzen 7 or greater
  • Memory: 16 GB RAM or more
  • Graphics Card: NVIDIA GeForce RTX 20-series / RTX 30-series or greater

Important Notes

  • Supported GPUs: Meta Quest Link supports a range of NVIDIA and AMD GPUs. Please check Meta's official documentation for the complete list of supported graphics cards.
  • Not Supported: Intel Arc GPUs are currently not supported by Meta Quest Link (support planned for future updates). NVIDIA RTX 2050, 3050, and 3050ti, as well as AMD Radeon RX 6500, are not recommended.

Connection Requirements

  • For Link (Wired): USB-C 3.2 cable that supports minimum 5 Gbps data transfer. Cable should be at least 3 meters (10 feet) for optimal comfort.
  • For Air Link (Wireless): Wi-Fi network (5GHz preferred) with strong signal strength
  • Software: Meta Quest Link PC app (free download from Meta)
  • Storage: At least 4 GB free space on PC for Meta Quest Link app installation
