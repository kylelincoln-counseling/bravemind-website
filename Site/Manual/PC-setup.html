This guide covers the initial setup and optimization of your Windows PC to ensure it is ready for a stable BRAVEMIND Ukraine session.

1. PC System Requirements

Before beginning, ensure your computer meets the following specifications. The recommended components are listed below. A system meeting the minimum requirements (Intel i5-4590, 8GB RAM, NVIDIA GTX 970) may work but is not officially supported for the best performance.

Recommended PC Components

  • Processor: Intel® Core™ i7 or Intel® Core™ i9
  • Graphics: NVIDIA® GeForce RTX™ 3060 or higher (e.g., NVIDIA® GeForce RTX 4050)
  • Memory: 16 GB
  • USB Port: 1x USB 3.0 port
  • Operating System: Windows 10 or Windows 11

Example Recommended Model: MSI Bravo 15 C7VE / 15.6" (1920x1080) IPS / Intel Core i7-11800H (8 (16) core 2.3 - 4.6 GHz) / 16 GB DDR5 / 512 GB SSD / nVidia GeForce RTX 4050, 6 GB GDDR6, 96-bit

2. PC Setup and Optimization

Follow these steps to configure your PC for optimal VR performance.

Install Meta Quest Desktop App

Download and install the Meta Quest setup software from the official Meta Quest website. Log in with your Meta account. This app is required to connect your headset to the PC.

Check for Meta Quest Link Driver Updates

After installing, open the Meta Quest Link app. If you see a notification that 'New drivers are required', click 'Update Now' and follow the prompts. After updating, you should see the standard home view.

Update Graphics Drivers

Open the NVIDIA App (or GeForce Experience) from your desktop. You can download it from the official NVIDIA website. Go to the Drivers tab, select the 'Game Ready Driver', and click Download, then Install.

Configure NVIDIA Control Panel

Right-click on your desktop and select NVIDIA Control Panel.

  1. Go to Manage 3D settings.
  2. Under 'Preferred graphics processor', select High-performance NVIDIA processor.
  3. In the settings list, find Power management mode and change it to Prefer maximum performance.
  4. Click Apply to save your changes.

Configure Windows Power Settings

Go to Windows Settings > System > Power & sleep > Additional power settings. Select the High performance power plan. For laptops, ensure the power mode is set to Best performance.

Configure Windows Display Settings

Right-click your desktop and select Display settings. Ensure Scale is set to 100% (Recommended) and Display resolution is set to the (Recommended) setting for your monitor. If using multiple monitors, set them to Extend these displays.

3. BRAVEMIND Ukraine Application Installation

Once your PC is optimized, install the latest BRAVEMIND software build.

  1. Download the latest build (9-26-24) from this link: BRAVEMIND Ukraine (9-26-24 Build).
  2. Create a new folder on your PC (e.g., C:\BRAVEMIND_Ukraine).
  3. In your file explorer, go inside the downloaded .zip file.
  4. Select all files and folders inside (You can use the hotkey Ctrl+A).
  5. Go to the folder you created earlier and paste everything there (Hotkey Ctrl+V).
  1. Locate the main application file, BraveMind Ukraine.exe.
  2. The first time you run the .exe, Windows Defender might block it. If this happens, click More info and then Run anyway.
  3. (Recommended) Right-click on BraveMind Ukraine.exe, select 'Show more options', then 'Send to' > 'Desktop (create shortcut)'. This will make it easier to launch in the future.

Your PC is now fully set up. The next step is to configure the Meta Quest headset and software.
