Common Issues

VR Headset Not Connecting

If the Meta Quest 2 headset is not recognized by the PC, please follow these steps...

  1. Ensure the Oculus Link cable is securely plugged into both the headset and a USB 3.0 port on the PC.
  2. Verify that 'Unknown Sources' is enabled in the Meta Quest Link desktop application settings.
  3. Restart both the PC and the Meta Quest 2 headset.
  4. If Meta Quest Link is not working, try resetting the Meta Quest headset (hold the power button and select 'Restart').

Application Fails to Launch

If the BRAVEMIND Ukraine application does not start, check the following...

  • Ensure 'Unknown Sources' is enabled (see 'First Time PC Setup').
  • Make sure the Meta Quest Link connection is active and you can see the PC VR environment in the headset.
  • Right-click the BraveMind Ukraine.exe and select 'Run as administrator'.
